# DeployHub Community

Welcome to the public community repository for DeployHub.

This repository is the primary place for:
- Reporting bugs
- Requesting features
- Sharing integration requirements
- Discussing enterprise onboarding workflows
- Providing feedback on deployment notifications and integrations

> 📌 Note: This repository does **not** contain the DeployHub core source code.
> The production system is maintained in a private repository.

---

## 🧭 What is DeployHub?

DeployHub is a multi-tenant platform used by teams to:
- Track deployments across environments
- Attach release notes to changes
- Send notifications via Slack and webhooks
- Collaborate with internal and external stakeholders during rollouts

---

## 📣 How to Use This Repository

### 1. Report Issues

Use Issues to report:
- Bugs in deployment tracking or notifications
- Problems with integrations (Slack, webhooks, etc.)
- Unexpected behavior during rollout workflows

Please include:
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or screenshots (if safe to share)

---

### 2. Request Features

We welcome feature requests, especially for:
- New integrations
- Improvements to notification workflows
- Enterprise onboarding and compliance features

---

### 3. Share Integration Requirements

Enterprise users often have specific requirements for:
- Security reviews
- Deployment approvals
- External stakeholder communication (e.g., Slack Connect)

Feel free to open an issue describing your workflow.
